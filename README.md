# grpc-micro

This project contains two services: **producer** and **consumer**.

---

## Run Locally

### Start Producer Service

```bash
cd producer
npm i
npm run start:dev
```

### Start Consumer Service (After the producer is running)

```bash
cd producer
npm i
npm run start:dev
```

### Or run with Docker Compose

```bash
docker compose build --no-cache
docker compose up
```
